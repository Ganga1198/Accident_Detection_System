# 🚨 Accident Detection & Alert System

An IoT-based smart accident detection and emergency alert system using ESP32, NodeMCU, GPS, and GSM modules.  
This project automatically detects accidents and sends emergency alerts with live location details to predefined contacts for faster response and improved road safety.

---

## 📌 Features

✅ Automatic accident detection  
✅ Real-time GPS location tracking  
✅ Emergency SMS alert system  
✅ ESP32 & NodeMCU based implementation  
✅ Fast response during emergencies  
✅ Low-cost IoT solution

## ⚙️ Working Principle

1️⃣ Sensors continuously monitor vehicle conditions  
2️⃣ Accident detection logic identifies collision events  
3️⃣ GPS module fetches live location coordinates  
4️⃣ GSM module sends emergency SMS alerts  
5️⃣ Emergency contacts receive accident location instantly
